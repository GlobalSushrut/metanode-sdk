Installation Instructions

Install directly from PyPI:

pip install metanode-sdk

Or install from the downloaded wheel:

pip install metanode_sdk-1.0.0b0-py3-none-any.whl

Getting Started

After installation, you can use the command-line tools:

metanode --help

Or import the SDK in your Python code:

from metanode.admin.k8s_manager import K8sManager

# Initialize the K8s manager
k8s_manager = K8sManager()

# Generate a cryptographic key for node authentication
result = k8s_manager.generate_node_crypt_key()
print(f"Generated key ID: {result['key_id']}")

Key Features

Kubernetes Integration

Deploy and manage secure testnet and application nodes on Kubernetes clusters with cryptographic verification.

Zero-Knowledge Proofs

Create and verify zero-knowledge proofs for secure data aggregation without revealing private information.

IPFS Integration

Easily store and retrieve data from the InterPlanetary File System (IPFS) with built-in tools.

Wallet Management

Create and manage cryptocurrency wallets, transfer tokens, and stake for mining eligibility.
