Quick Install

Install directly from PyPI:

pip install metanode-sdk

Or clone and install from source:

git clone https://github.com/metanode/metanode-sdk.git
cd metanode-sdk
pip install -e .

Connect to Testnet Infrastructure

The SDK automatically connects to our public testnet infrastructure at 159.203.17.36.

Create a wallet to interact with the network:

metanode-wallet create

Get tokens for deploying nodes and applications:

http://159.203.17.36/faucet

Connect to the Infrastructure

The MetaNode testnet is a blockchain infrastructure that manages server and client nodes. First, deploy your own nodes that connect to this infrastructure:

# Deploy your own server node
metanode-cli node deploy --type server

# Deploy your own client node
metanode-cli node deploy --type client

Then deploy your applications on your connected nodes:

# Deploy your app
metanode-cli app deploy app-config.json

Console-Only Mode

MetaNode SDK runs entirely in console mode without UI dependencies. All operations can be performed via CLI tools:

  • metanode-wallet - Wallet management
  • metanode-miner - Mining operations
  • metanode-cli - Deployment and status
  • metanode-cloud - Cloud resource management
